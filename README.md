# Tango Systems Official Company Website

The official, production-ready corporate website for Tango Systems Pvt Ltd. This project showcases a seamless blend of modern, user-centered UI/UX layout design and clean, responsive front-end development.

---

##  UI/UX & Creative Assets
* **Corporate Visual Identity:** Designed and standard-mapped a cohesive color palette, modern typography hierarchy, and asset layouts reflecting the company's tech-startup identity.
* **Responsive Wireframing:** Engineered fluid component structures to ensure flawless presentation across mobile, tablet, and desktop viewports.
* **Asset Creation:** Integrated custom vector icons and promotional graphics designed using industry-standard tools.

---

##  Tech Stack & Implementation
* **Frontend:** Semantic HTML5, CSS3 Grid/Flexbox layout structures, and JavaScript animations.
* **Deployment:** Optimized code structure for production-ready speeds.
